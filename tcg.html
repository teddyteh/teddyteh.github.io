close

Description

A real-time trivia game for couples with over 1 thousand downloads within the first week. Gameplay is as simple as:

  1. Player 1 creates a game.
  2. Player 2 joins with a link generated by player 1.
  3. On separate devices, both players answer some questions. Scores will be based on each other's answers.
  4. Game starts! Both players answer questions at their own paces. First to finish will wait for the other party.
  5. Results

2 months solo project. Developed was:

  • a backend in Node.js coupled with Socket.IO for real-time communication
  • a web client in Vue.js
  • as well as a mobile client in Flutter

Things I'm proud of:

  • Real-time! One player creates a lobby and another joins - games are played live!
  • Mobile first! Join game links first checks if a user has the app version, otherwise they would be redirected to the web version.
  • Dockerised backend for easier development and deployment.
  • Setup a CI pipeline in AWS using CodeBuild, CodeDeploy, ECR and Elastic Beanstalk.
